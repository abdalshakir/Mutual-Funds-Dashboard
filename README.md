# Sarmaaya Mutual Funds Dashboard

A modern, responsive dashboard for displaying Pakistani mutual funds data from the Sarmaaya API.

## Features

- 🕌 Islamic fund identification with mosque icon
- 📊 Real-time fund data display
- 🎯 Advanced filtering (Shariah compliant, Risk profile)
- 📄 Server-side pagination
- 📱 Fully responsive design
- 🎨 Modern gradient UI with hover effects

## Deployment

Deployed on Netlify for fast, global CDN distribution.

## Tech Stack

- Pure HTML/CSS/JavaScript
- Tailwind CSS (CDN)
- Sarmaaya REST API
